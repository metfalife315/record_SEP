💡 iPhone에서 파일 다운로드하기

iPhone에서 파일 다운로드가 안 될 때:
• Chrome 브라우저를 사용하세요!
• Chrome에서는 ZIP 파일 다운로드가 정상 작동합니다
• 녹음은 Safari, 다운로드는 Chrome이 최적입니다

📱 iOS User Guide:
• Use Safari browser for best compatibility
• Check Settings > Safari > Microphone permissions
• 💡 Chrome browser works better for file download!
• For file download: Use Chrome browser on iPhone
• For recording: Safari recommended
🤖 Android User Guide:
• Use Chrome browser for best compatibility
• Check Settings > Apps > Chrome > Permissions > Microphone
• Other browsers may have limited recording support
